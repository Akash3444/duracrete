Brick/Stone Color Grout

This is a cementatious/UV stable color mix used to fill joint spaces in wall Brick and Stone cladding instead of unprofessional ugly mortar( please refer to how to use manual and technical chart).

Antiquing Release Powder

This is a mix of premium hydrophobic powders used for smooth tool release in stamped concrete flooring. It also has a secondary function of adding an antique color to the primary color of the floated yet plastic concrete(please refer to how to use manual). It comes in 5 standard colors( almost night, Midnight, Dark earth, Mushroom and Nuetral). Customized color orders are also available at different rates.

Color Hardener

This is a dry shake cementatious/polymer fortified UV color powder used for the surface coloring and toughening of stamped and smooth concrete surfaces ( please refer to how to use manual). It comes in 18 standard colors. customized colors are also available at different rates (please check the color chart)

Brick/Stone Adhesive

This is a super bond cement/polymer based Adhesive used for the permanent fix of wall Brick/stone tile installation. It is a use alone product( please refer to how to use manual and technical chart)

Glazekote Sealant

This is a tough blend Acrylic solvent based sealant for stamped concrete floors. It is a semi-permeable tough floor seal used externally to seal and shine concrete( see how to use manual and technical chat).

Acrylic Putty

This is an acrylic based wall screed "ready to use" compound for a " non crack", "non puff", everlasting perfect smooth wall screeding( please refer to how to use manual and technical chart)
